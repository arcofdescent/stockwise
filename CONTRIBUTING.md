# Contributing to Stockwise

We're glad you're interested in contributing!

## Ways to Get Involved
- File bug reports or suggest new features
- Submit pull requests (code, docs, examples)
- Help improve docs, tests, or onboarding

## How to Get Started

1. **Fork & Clone**
2. **Set up local environment**
   - `backend`: Use `pyproject.toml` with uv or poetry
   - `frontend/package.json` for Node/React
3. **Follow coding style:** 
   - Python: Black, PEP8
   - JS: Prettier, StandardJS

### Pull Request Checklist

- Descriptive title and summary
- Reference relevant issues (if any)
- Pass existing and new tests
- Add/modify documentation as needed

## Community Guidelines

- Be respectful and constructive
- No unsolicited investment advice anywhere
- Do not share real trading credentials in public
